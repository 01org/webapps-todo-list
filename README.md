# INTRODUCTION
This application is written using HTML5/Javascript techology.

It has been tested on the following browsers/platforms:
* Tizen/WRT
* Tizen/Crosswalk
* Android/Crosswalk
* Tizen IDE
* Chrome/file
* Chrome/extension
